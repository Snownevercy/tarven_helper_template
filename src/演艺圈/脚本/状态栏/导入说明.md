# 状态栏脚本 - 导入说明

## 导入方式

### 方式一：本地开发（推荐用于测试和开发）

1. **启动本地开发服务器**

   在项目根目录运行：
   ```bash
   pnpm watch
   ```
   这将启动 webpack 监听模式，自动编译脚本到 `dist/演艺圈/脚本/状态栏/index.js`

2. **在酒馆助手中导入脚本**

   - 打开酒馆助手，进入脚本库
   - 点击"导入脚本"或"新建脚本"
   - 选择导入文件：`src/演艺圈/脚本/状态栏/导入到酒馆中/状态栏-本地开发.json`
   - 或者手动创建脚本，内容填写：
     ```typescript
     import 'http://localhost:5500/dist/演艺圈/脚本/状态栏/index.js'
     ```
   - 保存并启用脚本

3. **实时修改**

   启用"酒馆助手-实时监听-允许监听"开关后，修改 `src/演艺圈/脚本/状态栏/index.ts` 文件，脚本会自动热重载，无需刷新页面。

### 方式二：GitHub 自动更新（推荐用于生产环境）

1. **推送代码到 GitHub**

   ```bash
   git add src/演艺圈/脚本/状态栏/
   git commit -m "更新状态栏脚本"
   git push origin main
   ```

2. **等待 GitHub Actions 自动打包**

   - 推送后，GitHub Actions 会自动运行打包流程
   - 打包完成后，代码会自动更新到 `dist/` 目录
   - 通过 jsdelivr CDN 可以访问最新版本

3. **在酒馆助手中导入脚本**

   - 打开酒馆助手，进入脚本库
   - 点击"导入脚本"或"新建脚本"
   - 选择导入文件：`src/演艺圈/脚本/状态栏/导入到酒馆中/状态栏-GitHub自动更新.json`
   - 或者手动创建脚本，内容填写：
     ```typescript
     import 'https://testingcf.jsdelivr.net/gh/Snownevercy/tarven_helper_template@latest/dist/演艺圈/脚本/状态栏/index.js'
     ```
   - 保存并启用脚本

4. **自动更新**

   每次你推送代码到 GitHub 后，脚本会自动更新，无需手动重新导入。

## 注意事项

- **本地开发**：需要保持 `pnpm watch` 运行，且确保端口 5500 未被占用
- **GitHub 自动更新**：首次导入后，后续更新会自动生效，无需重新导入
- **实时监听**：启用后可以实时看到代码修改效果，非常适合开发调试
